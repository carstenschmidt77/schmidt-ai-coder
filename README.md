# Schmidt AI Coder

> Schmidt AI Code Extension - KI-Beratung, AI Agents und Embedded AI für effiziente Entwicklung. Forked from Schmidt AI Code.
>
> **AI Agents aus Hessen für Embedded Systems.**

- Generate code from natural language
- Checks its own work
- Run terminal commands
- Automate the browser
- Inline autocomplete suggestions
- Latest AI models (500+ supported)
- API keys optional

## Key Features

- **Code Generation:** Schmidt AI Coder can generate code using natural language.
- **Inline Autocomplete:** Get intelligent code completions as you type, powered by AI.
- **Task Automation:** Schmidt AI Coder can automate repetitive coding tasks to save time.
- **Automated Refactoring:** Schmidt AI Coder can refactor and improve existing code efficiently.
- **MCP Server Marketplace**: Schmidt AI Coder can easily find and use MCP servers to extend the agent capabilities.
- **Multi Mode**: Plan with Architect, Code with Coder, and Debug with Debugger, and make your own custom modes.

## Get Started

1. Install the Schmidt AI Coder extension from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=schmidt-embedded-systems.Schmidt-AI-Coder).
2. Configure your AI provider and API key.
3. Start coding with AI that adapts to your workflow.
